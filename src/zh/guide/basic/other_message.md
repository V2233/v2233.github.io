# 制作其它消息段

- 这里直接点点点就好，没有复杂的操作，需要注意的是这里所有添加的消息只作为一条消息发送，目的是可以图文混发，如果你同时添加语音和其它消息或是视频和其它消息类，将不被允许发送，具体实现取决于Yunzai的快速回复方法。如果您需要同一条指令发送多条消息，只要添加多条相同指令即可（这里我更愿意称作插件，未来将不会局限于message事件），您可以设置不同延时来调整发送顺序！
